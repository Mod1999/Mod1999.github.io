Инструкция по установке:
Видео-урок: https://www.youtube.com/watch?v=MX2-PAB9wrQ
Установка:
1)Залить файлы на хостинг
2)Зайти на http://вашсайт/install
3)Заполнить необходимые поля
 
Общие настройки:
1)зайдите в админ-панель (http://вашсайт/admin)
2)Выберите раздел "Настройки"
Название сайта - отвечает за заголовок cайта в меню и во вкладке
Фраза под заголовком - фраза в колонке приветствия на главной
Мета-описание сайта - используется для настройки meta description
Чтобы скрипт заработал вам нужно настроить данные Webmoney
   
1.Настройка оплаты WebMoney
WMID - ваш WMID
WMR - R кошелек 
WMZ - Z кошелек
Пароль от ключ-файла
Ключ-файл - полученный в Keeper Classic
Не знаете что такое ключ-файл и где его получить? Прочитайте инструкцию по этой ссылке: 
https://wiki.webmoney.ru/projects/webmoney/wiki/%D0%A4%D0%B0%D0%B9%D0%BB_%D0%BA%D0%BB%D1%8E%D1%87%D0%B5%D0%B9
Нажимайте "Сохранить"

2.Настройка оплаты через Яндекс.Деньги
Первым делом нужно зарегестрировать приложение в Яндексе
https://sp-money.yandex.ru/myservices/new.xml
Название вашего приложения - любое
Адрес вашего сайта - в формате (http://вашсайт.ру)
!!! Redirect URI - http://вашсайт.ру/yandex/token (например http://socday.3dn.ru/yandex/token)
Использовать проверку подлинности приложения - не ставьте галочку!
Вы получите Client ID
Перейдите по адресу http://вашсайт.ру/yandex/fix_yad
Потом по адресу по адресу http://вашсайт.ру/yandex/
Вас попросят ввести Client ID
Его нужно ввести в настройки вашего скрипта (http://вашсайт.ру/admin/config)
Выполнив это, опять переходите по адресу http://вашсайт.ру/yandex
Следуйте указаниям

3.Настройка оплаты через Qiwi
Зайдите в настройки скрипта
Укажите свой кошелек QIWI и пароль от него.
